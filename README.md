# TampermonkeyUserscripts

## 事前安裝設定

1. 安裝 [Tampermonkey](https://www.tampermonkey.net/) 瀏覽器擴充套件。
2. 右鍵點擊瀏覽器工具列的 Tampermonkey 圖示，選擇「管理延伸模組」。
3. 勾選「允許使用者指令碼」。
